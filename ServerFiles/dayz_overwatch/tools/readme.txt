Use db_utility.bat to clean up damaged vehicles and out of bound vehicles
Use db_spawnvehicles.bat to spawn vehicles.
Remember to got inn and set the password and username in the bat files, also put in your database name as well.
You will need to install http://strawberryperl.com/ but sure to get Strawberry Perl 32 Bit (NOT 64 Bit).
This is all based on DayZ Reality Private Server: https://github.com/thevisad/DayZ-Private-master
So threes a lot of information there about how to set this up.
